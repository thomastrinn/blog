Set Up File Server With Samba

March 7, 2022

A Samba segítségével fájl megosztást lehet lehetővé tenni Linux, Windows és MacOS operációs rendszerek számára.

Meglepetésemre a telepítés és beállítás sokkal egyszerűbb volt, mint gondoltam.

Az ubuntu oldalán található két leírás nagy segítséget adott:

  • Tutorial: Install and Configure Samba

  • Server - Docs: File Server

Nincs más teendő, mint telepíteni a samba csomagot:

sudo apt install samba

A beállításokat pedig az /etc/samba/smb.conf fájlban tehetjük meg.

A fájl alapértelmezett értékeit meghagytam, csupán a fájl végére felvettem a megosztásra szánt könyvtár beállításait:

[data]
   comment = Data
   path = /data
   guest ok = no
   read only = no
   browsable = yes
   create mask = 0755
   directory mask = 0755

  • [data]: a megosztás data néven lesz elérhető

  • comment: rövid leírás a megosztásról

  • path: az útvonal a megosztani kívánt könyvtárhoz

  • guest ok = no: jelszó nélkül nem érhető el a megosztás

  • read only = no: írási jog engedélyezve a megosztásra

  • browsable = yes: Windows kliensek számára engedélyezi a böngészést Windows Explorer-rel.

  • create mask = 0755: az újonnan létrehozott fájlok jogosultságait határozza meg.

  • directory mask = 0755: az újonnan létrehozott könyvtárak jogosultságait határozza meg.

alapértelmezetten a mask értékek 0700, biztonsági okokból, a 0755-el group=rw jogosultság adható meg.

A beállítások érvényesítése érdekében újra kell indítani a Samba service-t:

sudo service smbd restart

Ezt követően a felhasználóhoz rögzíteni kell egy Samba jelszót:

sudo smbpasswd -a username

A használt username-nek egy létező felhasználónak kell lennie.

Ezt követően már csatlakozhatunk a megosztott könyvtárhoz.

Linux és Mac esetén a fájl kezelőben a Connect to Server lehetőség alatt az alábbit megadva:

smb://ip-address/data

Windows esetén a File Manager-ben a fájl útvonal mezőben megadva:

\\ip-address\data
