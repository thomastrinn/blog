SonarQube in Docker

August 6, 2021

A SonarQube egy kód minőség elemző eszköz.

SonarQube futtatása docker-compose service-ként#

docker-compose.yml:

version: "3"

services:
  sonarqube:
    image: sonarqube:8.9-community
    container_name: sonarqube
    depends_on:
      - db
    ulimits:
      memlock:
        soft: -1
        hard: -1
      nofile:
        soft: 65536
        hard: 65536
    environment:
      SONAR_JDBC_URL: jdbc:postgresql://db:5432/sonar
      SONAR_JDBC_USERNAME: sonar
      SONAR_JDBC_PASSWORD: sonar
    volumes:
      - sonarqube_data:/opt/sonarqube/data
      - sonarqube_extensions:/opt/sonarqube/extensions
      - sonarqube_logs:/opt/sonarqube/logs
    ports:
      - "9010:9000"
  db:
    image: postgres:12
    container_name: sonarqube-postgres
    environment:
      POSTGRES_USER: sonar
      POSTGRES_PASSWORD: sonar
    ports:
      - 14432:5432
    volumes:
      - postgresql:/var/lib/postgresql
      - postgresql_data:/var/lib/postgresql/data

volumes:
  sonarqube_data:
  sonarqube_extensions:
  sonarqube_logs:
  postgresql:
  postgresql_data:

SonarQube indítása: docker-compose up -d

Mivel a SonarQube embedded Elasticserch-t használ szükség lehet a host gépen a következő beállításra:

sysctl -w vm.max_map_count=524288

Ha még így is gond volna, akkor a SonarQube Docker Hub oldalból induljál ki.

Sikeres indítás követően az alkalmazás elérhető a http://localhost:9010/ url-en.

Belépni admin/admin segítségével lehet.

Sonar Scanner futtatása docker container-ből#

Ha sonar-scanner-t kell futtatnunk a projektünk forráskódján, akkor azt is megtehetjük docker container-ből, az alábbi módon:

docker run --rm \
           --net host \
           -v "$(pwd):/usr/src" \
           sonarsource/sonar-scanner-cli \
           -Dsonar.projectKey=<your-projectKey> \
           -Dsonar.host.url=<your-sonar-host> \
           -Dsonar.login=<your-login>

A -v "$(pwd):/usr/src" azt a mapping-et mutatja, hogy ha a forráskódunk könyvtárában adjuk ki ezt a parancsot, akkor az a container /usr/src könyvtárára mutat, amelyben a sonar-scanner keresi a forráskódot.

A -Dsonar.* paraméterek értékét meg a sonarqube oldal a project létrehozásakor megad nekünk, s azokat kell ide behelyettesítenünk.
