Mounting a File System

April 22, 2022

Linux és UNIX operációs rendszerek esetén a mount parancs segítségével csatolhatunk fel fájlrendszereket. A felcsatolás helyét mount point-nak nevezik és a könyvtár struktúra egy tetszőlegesen választott helye.

Az unmount parancs segítségével pedig lecsatolható az adott fájlrendszer.

Fájlrendszer csatolása#

Az elérhető eszközök listázására használhatjuk az fdisk parancsot:

sudo fdisk -l

Ezzel részletes információkat kapunk az eszközökről és a partíciókról, illetve milyen fájlrendszerűek a partíciók.

Adott fájlrendszer csatolásának módja a mount paranccsal:

mount [OPTION...] DEVICE_NAME DIRECTORY

Konkrét példával, ha a /dev/sdb1 fájlrendszert akarjuk a /mnt/media könyvtárba csatolni, akkor azt a következőképp tehetjük:

sudo mount /dev/sdb1 /mnt/media

Amennyiben gyakori fájlrendszert, például ext4 vagy xfs csatolunk, akkor azt a mount fel tudja ismerni. Egyébként explicit meg kell határoznunk a fájlrendszer típusát. Ehhez a -t kapcsolót használhatjuk:

mount -t TYPE DEVICE_NAME DIRECTORY

További lehetőségek a -o mount options kapcsolók segítségével adhatóak meg:

mount -o OPTIONS DEVICE_NAME DIRECTORY

Fájlrendszer csatolása /etc/fstab segítségével#

Az /etc/fstab konfigurációs fájl tartalmazza a fájlrendszerek csatolásához szükséges információkat, ennek segítségével automatikussá tehető a fájlrendszerek csatolása.

A fájl tartalma több sor, melyek szerkezete az alábbi:

[Device] [Mount Point] [File System Type] [Options] [Dump] [Pass]

Pontos és részletes leírás az fstab-ról az ubuntu wiki Fstab oldalán található.

Egy gyakori beállítás:

UUID=<uuid> <pathtomount> <filesystem> defaults 0 0

Érdemes device-ként annak UUID -ját használni, amit az lsblk parancs segítségével tudhatunk meg:

sudo lsblk -o NAME,FSTYPE,UUID

Az UUID használata azért előnyös, mert ha a device neve változik is, attól függetlenül meg tud történni az eszköz csatolása.

Fontos a [PASS] értékének helyes megválasztása, mert ez határozza meg az fsck sorrendjét. Ennek részletei is megtalálhatóak a fenti ubuntu wiki linken.
